In this tutorial, I will demonstrate how to bind data to the ListView control.
This is an important concept as often you will find situations where you might be getting your data from an external source and you need to show this list to the user as a list.
Just an FYI, it isn’t necessary to do this as a list view, you may use a table instead, or perhaps even an unordered list.

    So what next?
    Lets create a listview.!
    How do we create one, you ask?
    Err.. I missed to tell you the catch :P We need an item template to decorate this view of each item in the model that we are using.
    Lets quickly write the item template and get back to writing our ListView.

    [code]
    <div id="list-view-container" data-win-control="WinJS.UI.ListView" data-win-options="{ itemDataSource : Application.Data.Movies.movies.dataSource,
    itemTemplate : select('#list-view-item-template'),
    selectionMode : 'multi',
    layout : {type : WinJS.UI.ListLayout},

    tapBehavior : 'toggleSelect',

    }">

    </div> [/code]

    Remember, always write the item template for a ListView or FlipView Control before defining the control.
